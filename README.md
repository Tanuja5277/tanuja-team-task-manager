# Team Task Manager

A full-stack Team Task Manager application built using Angular, Node.js, Express.js, and MongoDB.

## Features

* User Authentication
* Admin & Team Member Roles
* Task Creation & Assignment
* Task Status Management
* Dashboard UI
* REST APIs
* Responsive Design

## Tech Stack

### Frontend

* Angular
* TypeScript
* Bootstrap / CSS

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Frontend

```bash
cd frontend
npm install
ng serve
```

### Backend

```bash
cd backend
npm install
npm start
```
